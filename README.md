# Rate

一个五星评分组件，要用的话直接复制Rate.jsx这个文件到你的项目内，然后import一下。


文档如下：

属性	                       说明	                                 类型	                           默认值
disabled	             只读，无法进行交互	                            boolean	                         false
character	             自定义字符（不支持emoji）                       text	                                 ★
count	                     star总数	                                  number	                        5
defaultValue	             默认值	                                 number	                               5
onChange	             选择时的回调	                              Function(number)	                            -
